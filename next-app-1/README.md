## How to Start

1. Run `npm install`
2. npm install tailwindcss@latest postcss@latest autoprefixer@latest @tailwindcss/postcss@latest --save-dev

3. Add this in postcss.config.js

module.exports = {
plugins: {
"@tailwindcss/postcss": {},
autoprefixer: {},
},
};

4. Run `npm run dev`
